Example Projects

TweetScan

TweetScan is a project application I built to combine some Python twitter scraping code into a React single page website. Enter any term into the app and it will pull the latest 2000 tweets that reference the keywork. The app backend then combs through the results to analyse the tweets for term frequency. The frontend then displays to the users the 20 most frequent terms, along with the most popular and recent tweets. The frontend is written in React while the application backend is written in Python. It is currently hosted using Heroku.
